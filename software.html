Enumeration of 2-level polytopes

Open source software in perl that enumerates all combinatorial types of 2-level polytopes. 2-level polytopes are interesting classes of polytopes in optimization.

sourcecode upon request

Practical volume aproximation

Open source software that provides algorithms for volume approximation of polytopes in high dimensions (a few hundred), utilizing geometric random walks such as hit-and-run.

volume_approximation@github.com

HeaDDaCHe: Hashed Dynamic Determinants for Convex Hull and other geometric computation

HeaDDaCHe is a open source software which implements the scheme of hashed dynamic determinantal predicates for faster geometric computation, such as convex hull and point location algorithms.

sourcecode upon request

ResPol: A software package for computing projections of resultant polytopes

ResPol is a open source software to compute a projection of the Newton polytope of the resultant of a given polynomial system, or its orthogonal projection along a given direction. The resultant is fundamental in algebraic elimination, optimization, and geometric modeling.

ResPol@sourceforge.net
